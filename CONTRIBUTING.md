# Contributing to DRep Delegator Feedback System

Thank you for your interest in contributing! This project is part of a Cardano Project Catalyst initiative by SIDAN Lab to improve communication between delegators and DReps.

---

## How to Contribute

### Reporting Issues

Found a bug or have a feature suggestion?

1. Check existing issues to avoid duplicates
2. Create a new issue with a clear description
3. For bugs: include steps to reproduce, expected vs. actual behavior, and your environment

### Contributing Code

1. **Fork and clone the repository**

   ```bash
   git clone https://github.com/YOUR-USERNAME/drep-delegator-feedback-system.git
   cd drep-delegator-feedback-system
   ```

2. **Create a branch**

   ```bash
   git checkout -b feature/your-feature-name
   ```

   Use prefixes: `feature/`, `fix/`, `docs/`, `refactor/`, `test/`

3. **Make your changes**

   - Write clear commit messages (`feat:`, `fix:`, `docs:`, etc.)
   - Add tests for new functionality
   - Follow existing code style

4. **Push and create a Pull Request**

   ```bash
   git push origin feature/your-feature-name
   ```

   Include in your PR:

   - Description of changes
   - Related issue numbers
   - Testing performed
